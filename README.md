# Limited-visibility-into-finance-team-wellbeing
Limited visibility into finance team wellbeing
Description
Finance leaders often lack structured, recurring visibility into how their teams are feeling across workload, collaboration, management support and overall wellbeing. Without this visibility, managers may rely too heavily on assumptions or occasional check-ins, making it harder to identify early signs of burnout, disengagement or workload imbalance before they become turnover, missed targets or reduced productivity.
Technology / tools
There are leading, widely used employee experience and employee listening platforms in the market, including Culture Amp, Qualtrics EmployeeXM, Workday Peakon Employee Voice, Microsoft Viva Glint, Lattice, 15Five, Leapsome and Workleap Officevibe. However, a finance team can start with a simpler and more flexible solution using tools that may already be available internally.
This challenge can be addressed with lightweight survey and dashboard tools such as Google Forms, Microsoft Forms, Typeform or SurveyMonkey for data collection; Google Sheets, Excel, SharePoint or OneDrive for data storage; and Tableau, Power BI, Looker Studio or Qlik Sense for visualization.
Automation opportunity
Create an automated quarterly wellbeing feedback loop. Team members complete a short recurring survey, and responses are automatically collected, structured and visualized in a dashboard by quarter, year, finance subarea, location and wellbeing category.
This allows managers to move from informal impressions to structured visibility, helping them identify trends before burnout, disengagement, workload imbalance or collaboration issues become more serious.
Expected impact
Improved visibility into team wellbeing, earlier detection of people-related risks, better leadership decisions and more consistent follow-up actions. Over time, this can support stronger collaboration, higher retention, better workload balance and a more resilient finance function.
Proposed work
Created the quarterly wellbeing survey in Google Forms.
Downloaded the Google Sheets response dataset as an Excel file to use as the Tableau input file.
Built the Tableau workbook with score by question, trend, location map and finance subarea heatmap.
Prepared the dashboard preview for leadership review.
Documented the refresh steps to update the dashboard each quarter.
